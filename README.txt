MERN Ecommerce - Full project (starter)
--------------------------------------
Backend: backend/
  - npm install
  - copy .env.example -> .env and edit (MONGO_URI, JWT_SECRET)
  - npm run seed   # optional - creates sample data and admin user (admin@example.com / admin123)
  - npm start

Frontend: frontend/
  - npm install
  - npm start

Notes:
  - Backend serves uploads from /backend/uploads (images).
  - API base: http://localhost:5000/api
